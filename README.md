# AI Review Intelligence Dashboard

An AI-powered dashboard that analyzes live Google Play Store reviews using sentiment analysis and LLM-based issue summarization.

## Users can do 
Users can explore detailed dashboards that provide insights such as sentiment distribution, version-wise issue tracking, hourly anomaly detection, and review activity patterns. The platform also integrates Large Language Models (LLMs) to automatically generate summaries of customer feedback, highlight critical problems, and identify the most frequently reported issues affecting user experience.

The system combines web scraping, backend APIs, machine learning models, AI-driven text analysis, and real-time chart visualizations to transform raw review data into meaningful business and product insights. It is designed to support faster issue identification, user feedback analysis, and data-driven decision making for applications and digital products

## Features

- Live Play Store review scraping
- Sentiment analysis using DistilBERT
- AI-generated issue summaries using Qwen 2.5
- Interactive analytics dashboard
- FastAPI backend
- Docker deployment support

## Technologies Used

- Python
- FastAPI
- Hugging Face Transformers
- DistilBERT
- Qwen 2.5
- Google Play Scraper
